<IntroSection>

## What is this?

Cards, like any container, improves scannability of a design, providing a separate space for a user to focus on a specific task.

</IntroSection>

<Section>

## Container

```jsx live
<Card csx={{ width: '1/2', margin: 4 }}>
  <Center csx={{ width: '100%', height: 250, bg: '$secondary' }}>
    Content
  </Center>
</Card>
```

</Section>
