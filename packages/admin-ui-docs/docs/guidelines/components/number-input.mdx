## Doâ€™s

1. Always use a default value. If you do not know which one to use, pick 1. This way the user is not forced to type something and then click the buttons.
2. The lean mode is intended to be used on lists, that have a high density of content, and where being able to type the number is not important. It is especially useful in mobile contexts or cramped containers.
3. Do not use for big numbers. Steppers make more sense for small numbers that might be adjusted with a few clicks.
4. Add the units, whenever applicable. Ex. kg, mÂ², days, etc.
